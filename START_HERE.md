📧 EMAILJS SETUP (DETAILED):
═════════════════════════════════════════════════════════════════════════════

1. Go to https://www.emailjs.com/
2. Sign up (free account)
3. Go to Dashboard
4. Click "Email Services" → Add Service
5. Select Gmail
6. Authorize your Gmail account
7. Copy your Service ID
8. Go to "Email Templates" → Create New Template
   - Template Name: contact_form
   - Variables: from_name, from_email, subject, message
9. Save template and copy Template ID
10. Edit contact.html (line ~355):
    emailjs.init('YOUR_PUBLIC_KEY')
    emailjs.send('SERVICE_ID', 'TEMPLATE_ID', {...})

Done! Contact form will send emails to your Gmail.
