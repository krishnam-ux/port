## 🛠️ CUSTOMIZATION OPTIONS

### Change Colors
Edit CSS variables in `premium-3d.css`:
```css
:root {
  --purple-primary: #a855f7;
  --purple-light: #d946ef;
  --accent-blue: #0ea5e9;
}
```

### Adjust Animation Timing
```css
--transition-fast: 0.2s;
--transition-normal: 0.3s;
--transition-smooth: 0.5s;
```

### Control Effect Intensity
- Reduce particle count
- Decrease blur amounts
- Shorten animation durations
- Simplify transforms
