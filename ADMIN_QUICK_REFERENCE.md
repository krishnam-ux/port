# Admin Portal - Quick Reference Guide

## 🎯 Access Your Admin Portal

### Method 1: Keyboard Shortcut (Fastest) ⚡
Press: **`CTRL + F12`** (Windows/Linux) or **`CMD + F12`** (Mac)

### Method 2: Direct URL
Navigate to: `yourportfolio.com/admin.html`

### Method 3: From Portfolio
The admin portal is hidden from the public navigation for security!

## 🐛 Common Issues & Solutions

### Can't Login?
- Verify email and password
- Check caps lock
- Clear browser cache
- Try different browser

### Admin Portal Won't Open?
- Try `CTRL + F12` keyboard shortcut
- Direct navigation to `/admin.html`
- Check URL is correct
- Refresh page and try again

### Messages Not Showing?
- Refresh the dashboard
- Check internet connection
- Wait for data to load
- Check browser console for errors

### Images Not Uploading?
- Check file size (should be reasonable)
- Verify file format (JPG, PNG, WebP)
- Check internet connection
- Try uploading single image first
