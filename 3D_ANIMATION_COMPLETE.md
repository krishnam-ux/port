## 🔧 USAGE & CUSTOMIZATION

### To Use the Effects:

1. **Scroll-Based Reveals**: Add `data-reveal` attribute to elements
   ```html
   <div data-reveal>This will fade in on scroll</div>
   ```

2. **Parallax Scroll**: Add `data-parallax-scroll` with speed value
   ```html
   <div data-parallax-scroll="0.5">Parallax element</div>
   ```

3. **Skill Cards**: Use `data-skill-level` for animated progress
   ```html
   <div class="skill-card" data-skill-level="85">Python</div>
   ```

4. **Floating Inputs**: Already enabled on contact form

### CSS Variables (Customizable):

```css
:root {
  --purple-primary: #a855f7;
  --purple-light: #d946ef;
  --transition-smooth: 0.5s cubic-bezier(0.4, 0, 0.2, 1);
}
```
