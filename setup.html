📦 Optional: Setup Firebase (For Production)

To switch from Demo mode to live Firebase cloud storage:

  1. Go to Firebase Console
  2. Create a new project (or use existing one)
  3. Enable Authentication (Email/Password)
  4. Create Firestore Database
  5. Copy your Firebase Config
  6. Open js/firebase-config.js
  7. Replace the firebaseConfig object with your credentials
  8. Add admin user to Firebase Authentication
  9. Refresh the page - it will auto-switch to Firebase mode
